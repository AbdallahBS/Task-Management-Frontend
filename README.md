## Setup Instructions

### Prerequisites
- Node.js (version X.X.X)
- npm (version X.X.X)

### Cloning the Repository
```bash
git clone <repository-url>
cd <repository-directory>
```

### Installing Dependencies
```bash
npm install
```

## Running the Project

To start the frontend server:
```bash
npm run dev
```
